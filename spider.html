Peacock Spider

This Figma prototype of a website that educates the general public about Peacock Spiders. The goal was to create an infographic but because I wanted to challenge myself to learn a new program, Figma, I decided to create a website prototype.

Continue reading to learn about the process that went into creating this piece.

Research of Potential Infographic Composition

Initially I had thought about doing a traditional infographic and therefore after researching the information I wanted to include, I sketched up a composition.

Modeled the Spider

Using Nomad Sculpt, I roughly modeled out the spider from a sphere. I did not have time to properly UV wrap, rig, animate the model, so I created gifs that could be uploaded onto Figma. Movement of the legs were essential to the storytelling of this piece so I wanted to make sure it came across.

Figma Prototype

The remaining illustrations were created via digital painting and an interactive map showing geographic spread of different species of peacock spiders was created via Tableau. All the assets were put together within Figma to create an interactive prototype.
